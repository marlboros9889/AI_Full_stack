# webapp 폴더 안내

이 프로젝트는 현재 Spring Boot + Thymeleaf 방식으로 화면을 렌더링합니다.

- 실제 고객 화면: `src/main/resources/templates/index.html`
- 실제 CSS/JS/이미지: `src/main/resources/static`
- JSP 방식으로 전환하거나 선생님 예제처럼 `webapp` 흐름을 설명해야 할 때 이 폴더를 기준으로 확장합니다.

현재 기능을 깨지 않기 위해 실행 화면은 Thymeleaf 위치에 유지하고, `webapp`은 JSP 전환 또는 발표 설명용 구조 기준으로 둡니다.
