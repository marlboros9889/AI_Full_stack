# 예약 서비스 기능 목록

## 현재 구현

| 기능 | 사용자 | 상태 |
|---|---|---|
| 시술 메뉴 조회 | 고객 | 구현 |
| 7일 이내 예약 가능 시간 조회 | 고객 | 구현 |
| 예약 생성·수정·취소 | 고객 | 구현 |
| 내 예약·이전 시술 이력 조회 | 고객 | 구현 |
| 예약 상태·휴무일·시술 관리 | 관리자 | 구현 |

## 다음 단계

| 기능 | 이유 | 우선순위 |
|---|---|---|
| 예약 확인·리마인드 알림 | 노쇼 감소 | 높음 |
| 관리자 캘린더 화면 | 일별 예약 운영 | 높음 |
| 고객 연락처 등록/수정 | 소셜 로그인 예약 이력 연결 | 높음 |
| 대기 예약 | 취소 시간 활용 | 중간 |
| 결제·쿠폰 | 운영 확장 | 낮음 |

## 제외

React, 결제 연동, 범용 OAuth 제공자, 마케팅 자동화는 현재 학습 범위를 넘으므로 기본 예약 CRUD가 안정화된 뒤에만 추가한다.
