1. block vs inline

block
width o , 줄바꿈 o
inline
width x , 줄바꿈 x

2. 배치

  • float : 왼쪽, 오른쪽 옆에 붙이기 (끊기:clear:both)
  • position : 정확한 자리를 정해서 붙이기 (좌표)
  • display : 블록에 줄서기, 속성 바꾸기
